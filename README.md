# Database Building Blocks

Material for the Database Building Blocks talk held at the remote meetup of Nairobi LUG on 3rd July 2021.

## Run the presentation locally:

1. Setup a Go environment: https://golang.org/doc/install
2. `go get -u golang.org/x/tools/cmd/present`
3. In this directory, run `present -base=$(go env GOPATH)/src/golang.org/x/tools/cmd/present/ -orighost localhost`
4. Visit http://localhost:3999 in your browser.

More on [present](https://godoc.org/golang.org/x/tools/cmd/present).
